# 🪨📄✂️ Rock Paper Scissors Game (Python)

An interactive **Rock Paper Scissors** game built using Python. This version includes **ASCII visuals**, **user input handling**, and a **replay system**, making it more engaging than a basic implementation.

---

## 🚀 Features

- 🎮 Play against the computer
- 🔁 Infinite gameplay loop (until user exits)
- 🎲 Random computer choices using Python
- 🎨 ASCII art representation for Rock, Paper, Scissors
- ❌ Input validation (handles invalid inputs)
- 🔄 Replay option after every round

---

## 🛠️ Technologies Used

- Python 3
- Built-in `random` module

## ⚙️ How It Works

1. The computer randomly selects a number between 1 and 3.
2. The user inputs:
   - `1` for Rock 🪨
   - `2` for Paper 📄
   - `3` for Scissors ✂️
3. The program:
   - Displays the user's choice with ASCII art
   - Displays the computer’s choice
4. The winner is decided using standard rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
5. The user is prompted to play again or exit.
